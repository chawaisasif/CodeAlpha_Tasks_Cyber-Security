# 🎣 Phishing Awareness Training

## 📌 Project Overview

This project is a **Phishing Awareness Training Presentation** designed to educate users about phishing attacks, their risks, identification techniques, and prevention strategies.

Phishing is one of the most common and dangerous cybersecurity threats today. This training helps users understand how attackers operate and how to stay protected.

---

## 🎯 Objectives

* Understand what phishing is
* Identify different types of phishing attacks
* Recognize phishing emails and fake websites
* Learn social engineering tactics
* Apply best practices to prevent attacks

---

## 🧠 What is Phishing?

Phishing is a **cyber attack** in which attackers pretend to be trusted sources (like banks or companies) to steal sensitive information such as:

* Login credentials
* Bank details
* Personal data

---

## ⚠️ Why Phishing is Dangerous

Phishing attacks can cause:

* Identity theft
* Financial loss
* Unauthorized account access
* Data breaches

---

## 🧪 Types of Phishing

* 📧 **Email Phishing** – Fake emails from trusted sources
* 📱 **Smishing** – Fraud messages via SMS
* 📞 **Vishing** – Scam calls impersonating authorities
* 🎯 **Spear Phishing** – Targeted attacks on individuals

---

## 📧 Email Phishing (Most Common)

Attackers send fake emails that:

* Look legitimate
* Ask for sensitive data
* Contain malicious links or attachments

### 🚩 Signs of Phishing Emails

* Unknown sender
* Urgent or threatening language
* Spelling/grammar mistakes
* Suspicious links
* Unexpected attachments

---

## 🌐 Fake Website Identification

| Real Website ✅      | Fake Website ❌        |
| ------------------- | --------------------- |
| HTTPS (🔒 secure)   | No HTTPS              |
| Professional design | Poor design           |
| Correct URL         | Slightly modified URL |
| Secure login        | Data stealing         |

**Example:**

* ✅ [https://www.bank.com](https://www.bank.com)
* ❌ [http://bank-login123.com](http://bank-login123.com)

---

## 🧠 Social Engineering Tactics

Attackers manipulate users using:

* ⚡ Urgency (“Act now!”)
* 😨 Fear (“Account will be blocked”)
* 🏢 Authority (“We are from your bank”)
* 🎁 Rewards (“You won a prize”)

---

## 🎭 How Attackers Trick Users

* Impersonate trusted organizations
* Create panic situations
* Use fake logos and branding
* Send convincing messages

---

## 🛡️ Prevention Tips

* Do not click unknown links
* Verify sender email address
* Use strong passwords
* Check website URLs carefully
* Enable Two-Factor Authentication (2FA)

---

## 🌐 Safe Browsing Practices

* Keep software updated
* Use antivirus/security tools
* Avoid public Wi-Fi for sensitive tasks
* Always check for HTTPS 🔒

---

## 🌍 Real-World Examples

* Fake PayPal login emails
* Google security alert scams
* Bank SMS verification scams
